

# 📝 To-Do Task Manager

## 📖 Overview

The **To-Do Task Manager** is a lightweight and user-friendly web application developed using **HTML, CSS, and JavaScript**.
This application helps users organize their daily activities by allowing them to add, complete, and remove tasks easily.
It focuses on improving productivity through a simple and clean interface.

---

## ✨ Key Features

* Add new tasks instantly
* Mark tasks as completed
* Remove tasks from the list
* Data persistence using **Browser Local Storage**
* Responsive design for mobile and desktop screens
* Smooth and minimal user interface

---

## 🧠 Concepts Implemented

* DOM Manipulation
* Event Listeners
* Array & Object Handling
* Local Storage API
* Responsive Web Design

---

---

## ▶️ How to Use

1. Clone or download the repository
2. Open the project folder
3. Run the **index.html** file in any browser
4. Start adding your daily tasks

---

## 📸 Sample Workflow

* Enter a task
* Click **Add**
* Mark it as **Completed** when done
* Delete it if no longer needed

---

## 🎯 Project Purpose

This project was built to strengthen **JavaScript fundamentals** and understand how real-time task management applications work in web development.
